第2个返回参数为拟合函数在点 `x=25:0.4:60` 处的函数值（用1个行向量表示）。

## 综合训练

请用 Matlab 函数 `ode23` 求解下列微分方程：

$$\begin{cases}
    \frac{d^2x}{dt^2}=20(1-x^2)\frac{dx}{dt}+0.5x\\
    x(0)=2;x'(0)=0
\end{cases}$$

编写函数调用 `ode` 工具箱函数返回 $x$ 在点 `0:0.1:100` 处的函数值，用列向量存储这些函数值,并绘制出函数 $x$ 在区间[0,100]上的曲线。此列向量为 `double` 型数组。
